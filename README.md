## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- Git
- PostgreSQL 14+

### Development Setup

1. **Clone Repository**
```bash
git clone https://github.com/AlloxLab/allox-core.git
cd allox-core
```

2. **Install Dependencies**
```bash
# Root dependencies
npm install

# Frontend dependencies
cd frontend
npm install
cp .env.example .env.local

# Backend dependencies
cd ../backend
npm install
cp .env.example .env
```

3. **Database Setup**
```bash
# Generate Prisma client
cd backend
npx prisma generate
npx prisma migrate dev
```

4. **Start Development Servers**
```bash
# Start backend (from backend directory)
npm run dev

# Start frontend (in new terminal, from frontend directory)
npm run dev
```

5. **Smart Contract Development**
```bash
cd contracts
npm install
cp .env.example .env
npx hardhat compile
```
